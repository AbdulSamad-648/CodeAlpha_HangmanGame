## Implementation

Words are stored in Python as a list:

```python
WORD_LIST = ['python', 'hangman', 'computer', 'programming', 'internship']
```

The game randomly selects one word using:
```python
word = random.choice(WORD_LIST).upper()
```

## Adding More Words

To expand the word list, modify the `WORD_LIST` variable in `hangman_game.py`:

```python
WORD_LIST = [
    'python',
    'hangman',
    'computer',
    'programming',
    'internship',
    # Add more words here
    'algorithm',
    'database',
    'network',
    # etc.
]
```
